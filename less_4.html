Урок 4: Память и Настройки (EEPROM)

Научимся сохранять данные так, чтобы они не пропадали после выключения питания.

💡 Что такое EEPROM?

Представьте, что у Arduino есть два типа памяти:

  • Оперативная память (RAM): Быстрая, но временная. Переменные живут здесь только пока есть питание. Выключили плату — данные стерлись.
  • EEPROM: Медленнее, но постоянная. Это как маленький жесткий диск внутри чипа. Данные хранятся годами даже без батареек.
⚠️ Важно!
Ячейку EEPROM можно перезаписать примерно 100 000 раз. Нельзя записывать данные в каждом цикле loop() (например, каждую миллисекунду), иначе вы сотрете память за пару дней. Записывайте только при изменении настроек!

🛠 Практика: «Диммер с памятью»

Мы сделаем устройство, которое регулирует яркость светодиода. Если вы выключите Arduino, а потом включите снова — она вспомнит последнюю яркость.

Схема подключения:

  • Светодиод: Пин 9 (через резистор 220 Ом).
  • Потенциометр: Средний пин на A0, боковые на 5V и GND.

Код программы:

#include <EEPROM.h>

// Пины
const int LED_PIN = 9;
const int POT_PIN = A0;

// Адрес в EEPROM (ячейка №0)
const int ADDR_BRIGHTNESS = 0;

int currentBrightness = 0;     // Текущая яркость
int savedBrightness = 0;       // Яркость из памяти

void setup() {
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);

  Serial.println("--- Запуск системы ---");

  // 1. ЧТЕНИЕ ИЗ ПАМЯТИ
  // Считываем байт по адресу 0
  savedBrightness = EEPROM.read(ADDR_BRIGHTNESS);
  
  Serial.print("Прочитано из памяти: ");
  Serial.println(savedBrightness);

  // Сразу устанавливаем сохраненную яркость
  analogWrite(LED_PIN, savedBrightness);
  currentBrightness = savedBrightness;
}

void loop() {
  // 2. ЧТЕНИЕ С ПОТЕНЦИОМЕТРА
  int potValue = analogRead(POT_PIN); 
  int newBrightness = map(potValue, 0, 1023, 0, 255);

  // 3. ПРОВЕРКА: Записываем ТОЛЬКО если значение изменилось
  if (newBrightness != currentBrightness) {
    
    currentBrightness = newBrightness;
    
    // Меняем яркость
    analogWrite(LED_PIN, currentBrightness);
    
    // 4. ЗАПИСЬ В ПАМЯТЬ
    EEPROM.write(ADDR_BRIGHTNESS, currentBrightness);
    
    Serial.print("Новая яркость: ");
    Serial.println(currentBrightness);
    
    delay(100); // Небольшая задержка
  }
}

Разбор ключевых функций:

  • EEPROM.read(адрес) — читает число от 0 до 255 из указанной ячейки.
  • EEPROM.write(адрес, значение) — записывает число в ячейку.
  • if (new != current) — эта проверка спасает память от преждевременного износа.

📝 Домашнее задание

Сделайте «Счетчик включений».

  1. При старте (setup) считывайте число из EEPROM.
  2. Прибавляйте к нему единицу (+1).
  3. Записывайте новое число обратно.
  4. Выводите количество включений в Serial Monitor.

Подсказка: Для чисел больше 255 используйте функции EEPROM.put() и EEPROM.get().
